# Print N Pack

A simple Next.js website for a packaging solution company.

## Setup

This project uses:
- Next.js 13.4.19
- React 18.2.0
- Tailwind CSS 3.3.3
- Node.js 18.x

## Development

1. Make sure you're using Node.js 18.x
   ```
   nvm use
   ```

2. Install dependencies:
   ```
   npm install
   ```

3. Run the development server:
   ```
   npm run dev
   ```

## Deployment

1. Build for production:
   ```
   npm run build
   ```

2. Start production server:
   ```
   npm start
   ```
